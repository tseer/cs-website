Director(s): [Tiago Cogumbreiro]({{ '/people/tiago-cogumbreiro/' | relative_url }})

The aim of the Software Verification Laboratory is to improve programmer productivity by developing tools and techniques that localize bugs and enforce the correctness of systems. Our main focus is on verifying scientific codes in the domain of high performance computing. We employ the fields of programming languages, formal methods, concurrency theory, and statistical machine learning to improve software correctness.
