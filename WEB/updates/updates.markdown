This page highlights ongoing improvements to the UMass Boston Computer Science website.

## Member 1 – Austin
### Improved Structured Data for AI Features

**Description:**
- Fix broken links across the site  
- Add structured data (JSON-LD) for courses and pages  
- Improve keywords and summaries for AI readability  
- Enhance internal linking between courses, programs, and resources  

**Why it’s needed:**
AI-assisted systems can hallucinate when content is unstructured or inconsistent.  
Improving structured data ensures:
- Better search accuracy  
- More reliable AI responses  
- Stronger SEO and machine readability  

**Current Progress:**  
- Structured course pages implemented  
- ItemList schema added for courses  
- Ongoing improvements to programs/pages

- Map Jekyll collections (news, research, events)
- Enable GitHub authentication for safe commits

**Why it’s needed:**
- Simplifies content updates for non-technical users  
- Reduces manual editing of markdown files
